For array - 
if contiguous array sum, and constraints contain negative numbers - kadane
O(N) constraint, sort, in-place, N objects - DNF
Majority Element, appear more than n/2 times - moore's voting
Subarray, Substring, expand/shrink - sliding window
Sorted array, find pair, sum - two pointers
Range Sum - Prefix sum
Range Update - Difference array

For Stack-
Next greater/smaller element - monotonic stack
Parentheses, valid brackets, matching pairs
Undo, History tracking
Min/Max Stack find min/max in O(1) time  

For Queue - 
BFS/Level Order traversal, level by level, depth
Sliding window Min/Max
FIFO tracker
PriorityQueue/Heap Top k element, Kth largest, merge k sorted

For LinkedList - 
Fast & slow pointers
In-Place reversal 
Dummy Node trick to not loose the track of head while performing operations
Intersection trick, just replace the nodes after reaching null

For Trees:
DFS depth explorer
BFS layer/level scanner
Post-Order aka Bottom Up recursion, when parent node can't make decision without children LCA
